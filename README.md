# JavaScript

This repository contains my JavaScript learning practice files.

## Contents

- `1-intro/` - Basic JavaScript setup, alerts, console output, prompts, and confirmation dialogs.
- `2-variables,data-types/` - Variables, `var`, `let`, `const`, primitive data types, and object basics.
- `3-conditional-statements/` - Conditional statements, operators, comparisons, logical operators, and ternary expressions.
- `4-loops/` - `for`, `for...in`, `for...of`, `while`, and `do...while` loops.
- `5-functions/` - Function declarations, parameters, return values, default parameters, and arrow functions.
